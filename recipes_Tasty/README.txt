recipe_numbers.json = Počet receptov ku každému jedlu

recipe_names.json = ID receptov ku každému jedlu (82 jedál)

recipe_names_min_10.json = ID receptov k jedlám, u ktorých máme aspoň 10 receptov (38 jedál)
recipe_names_min_15.json = ID receptov k jedlám, u ktorých máme aspoň 15 receptov (24 jedál)
recipe_names_min_20.json = ID receptov k jedlám, u ktorých máme aspoň 20 receptov (4 jedlá)


Každý podadresár je nazvaný podľa ID jedla a obsahuje ku každému receptu:
  ID_RECEPTU.jpg = Obrázok
  ID_RECEPTU.json = Štruktúrované dáta o recepte včetne zoznamu ingrediencií ("recipeIngredient") a zoznamu inštrukcií ("recipeInstructionSteps").


Recepty, ktoré neobsahovali žiadny text, sú už odstránené.
